# 📊 Analytics Data Folder

วางไฟล์ CSV ที่ดาวน์โหลดมาจากหลังบ้าน YouTube Studio หรือ Facebook Creator Studio ที่นี่แก!

- **สถิติการดูต่อเนื่อง (Audience Retention):** แนะนำให้ตั้งชื่อไฟล์ทำนอง `youtube_retention.csv`
- **สถิติวิดีโอรวม (Video Metrics):** แนะนำให้ตั้งชื่อไฟล์ทำนอง `youtube_metrics.csv`
