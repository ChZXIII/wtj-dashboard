# คู่มือการตั้งค่า Facebook Auto-Post (ดึงคีย์ Access Token) 🔑🤖

ยินดีต้อนรับสู่ขั้นตอนปูสายเคเบิลระบบสื่อสารอัตโนมัติของทีมงานเราจ้า! เมื่อแกทำตามคู่มือนี้เสร็จ แกจะได้คีย์ความปลอดภัย 2 ตัวมาใส่ไว้ในไฟล์ `.env` ของโปรเจกต์ ซึ่งจะทำให้สคริปต์ [facebook_publisher.py](../skills/facebook_publisher.py) ทำงานได้ทันทีจ้า!

---

## 🏁 สิ่งที่แกต้องมีก่อนเริ่ม:
1. บัญชี **Facebook ส่วนตัว** ปกติที่แกใช้งาน
2. **Facebook Page** ที่แกเป็น "ผู้ดูแลระบบ (Admin)"

---

## 📝 Step 1: สมัครและสร้างแอปบน Facebook Developer (ระบบใหม่ 2025)
1. เข้าไปที่เว็บ [Facebook Developers](https://developers.facebook.com/) แล้วล็อกอินด้วยบัญชี Facebook ของแก
2. หากยังไม่เคยเป็นนักพัฒนา ระบบจะป๊อปอัปให้กดสมัคร (Get Started / Register) ให้กดยอมรับและสมัครจนเสร็จจ้า
3. พอถึงหน้าหลัก (My Apps) ให้กดปุ่มสีเขียว **`Create App`** (สร้างแอปพลิเคชัน)
4. หน้ารายละเอียดของแอพ: ตั้งชื่อแอปในช่อง **ชื่อแอพ** (เช่น `WTJ Veda Bot`) ➡️ กด **ถัดไป (Next)**
5. หน้ากรณีการใช้งาน (Use cases): ให้เลือกแท็บ "การจัดการเนื้อหา" หรือ "ทั้งหมด" แล้วเลือก 👉 **"จัดการทุกอย่างบนเพจของคุณ" (ที่มีไอคอนรูปธงสีเทา)** ➡️ กด **ถัดไป**
6. หน้าธุรกิจ (Business): เลือกพอร์ตโฟลิโอธุรกิจของแก (ถ้ามี) หรือข้ามไปก่อนถ้าไม่มี ➡️ กด **สร้างแอพ** (และกรอกรหัสผ่าน Facebook เพื่อยืนยัน)

---

## 🔗 Step 2: แอดระบบล็อกอินเพื่อทะลวงบั๊ก Graph API (สำคัญมาก!)
1. พอแอปสร้างเสร็จและเข้ามาที่หน้า **แดชบอร์ด** ของแอปแล้ว
2. ให้เลื่อนหน้าจอลงมาด้านล่างๆ หาหัวข้อ 🌟 **"เพิ่มผลิตภัณฑ์" (Add Products)**
3. มองหาผลิตภัณฑ์ที่ชื่อว่า **"การเข้าสู่ระบบด้วย Facebook" (Facebook Login)** ➡️ แล้วกดปุ่ม **"ตั้งค่า" (Set up)** 
*(กดแค่ปุ่มตั้งค่าให้มันแอดเข้าแอปก็พอ ไม่ต้องไปกรอกอะไรเพิ่มเติมจ้า)*

---

## 🔑 Step 3: ดึงรหัสผ่านลับ (Personal Page Access Token)
นี่คือขั้นตอนดึงคีย์จริงผ่านเครื่องมือด่วนของ Facebook จ้า:
1. เปิดแท็บใหม่ เข้าไปที่เครื่องมือทดสอบระบบของ Facebook: 
   🔗 **[Graph API Explorer](https://developers.facebook.com/tools/explorer/)**
2. สังเกตแถบเมนูด้านขวามือของจอ:
   - ตรงช่อง **`Meta App`** เลือกชื่อแอปที่แกเพิ่งสร้าง (เช่น `WTJ Veda Assistant`)
   - ⚠️ **คำเตือนสุดสำคัญ:** ตรงช่อง **`User or Page`** (ผู้ใช้หรือเพจ) ปล่อยทิ้งไว้คำว่า "รับโทเค็น" โง่ๆ แบบนั้นเลย! **ห้ามไปกดพยายามเลือกอะไรในนี้เด็ดขาด** (เดี๋ยวระบบมันจะทำให้เองอัตโนมัติ)
3. เลื่อนลงมาตรงหัวข้อ **`Permissions`** (สิทธิ์การอนุญาต) ให้แกเลือกสิทธิ์ 3 ตัวนี้ใส่เข้าไป (กดลูกศรขวาสุดของช่องพิมพ์แล้วเอาเมาส์คลิกเลือกเลยจ้า แก้บั๊กพิมพ์ไม่ติด):
   - `pages_manage_posts`
   - `pages_read_engagement`
   - `pages_show_list`
4. พอเลือกสิทธิ์ครบ 3 ตัว ➡️ ให้แกกดปุ่มสีฟ้าเบ้อเริ่ม **`Generate Access Token`** เลยจ้า!
5. หน้าต่าง Facebook จะเด้งขึ้นมาให้กดยืนยันตัวตน (กดยอมรับให้หมด) พอกลับมาที่หน้าเดิม... ระบบจะฉลาดพอที่จะเปลี่ยนช่องผู้ใช้เป็น "รับโทเค็นการเข้าถึงของผู้ใช้" ให้เองอัตโนมัติโดยที่เราไม่ต้องเหนื่อย!
6. 🛑 **หยุดก่อน! อย่าเพิ่งรีบเลือกเพจ:** ตอนนี้รหัสในช่องบนสุดคือ "คีย์ผู้ใช้แบบสั้น" ให้แก **ก๊อปปี้รหัสคีย์ผู้ใช้นี้เก็บไว้** ก่อนเลยจ้า!

---

## 🔒 Step 4: แปลงร่างคีย์เพจให้เป็น "อมตะ" (Never Expires)
กฎเหล็กของ Facebook: เราต้องเอาคีย์ผู้ใช้ไปขยายเวลาก่อน แล้วค่อยเอากลับมาดึงคีย์เพจ ถึงจะได้คีย์อมตะ!

1. เปิดแท็บใหม่ เข้าไปที่เครื่องมือตรวจสอบโทเค็น:
   🔗 **[Access Token Tool / Debugger](https://developers.facebook.com/tools/debug/accesstoken/)**
2. เอา "คีย์ผู้ใช้แบบสั้น" ที่เพิ่งก๊อปมา วางลงในช่องค้นหา ➡️ กดปุ่ม **`Debug`**
3. เลื่อนลงมาล่างสุด หาปุ่มที่เขียนว่า **`Extend Access Token`** (ขยายเวลาใช้งาน) ➡️ กดปุ่มนั้น 
4. แกจะได้รหัสคีย์ผู้ใช้ตัวใหม่ที่อายุยาวขึ้น (60 วัน) โผล่ขึ้นมา ➡️ **ให้ก๊อปปี้คีย์ 60 วันตัวใหม่นี้ไว้!**
5. 🏃‍♂️💨 กลับมาที่แท็บ **Graph API Explorer** หน้าเดิม
6. เอาคีย์ 60 วันที่เพิ่งก๊อปมา **"วางแปะทับ"** รหัสเดิมในช่องบนสุด (ช่องโทเค็นการเข้าถึง)
7. 🌟 คราวนี้แหละ! ให้แกคลิกที่ช่อง **`User or Page`** ➡️ แล้วเลือกเมนูย่อย **`รับโทเค็นการเข้าถึงเพจ` (Get Page Access Token)**
8. จิ้มเลือกชื่อเพจของแก ➡️ รหัสช่องบนสุดจะเปลี่ยนไปอีกรอบ... คราวนี้มันคือ **Page Access Token แบบอมตะ (Never Expires)** ของจริง!! ก๊อปปี้เก็บไปใช้ได้ตลอดชาติเลยจ้า! (อย่าลืมก๊อป **`Page ID`** ที่อยู่ด้านล่างด้วยนะ)

---

## 📂 Step 5: นำคีย์มาแปะในไฟล์ `.env` บนเครื่องแก!
1. สร้างไฟล์ชื่อ **`.env`** ไว้ในโฟลเดอร์หลักของโปรเจกต์ (โฟลเดอร์เดียวกับที่มี `README.md` อยู่จ้า)
2. เขียนข้อความลงไปในไฟล์ตามรูปแบบนี้จ้า (เอาคีย์และ Page ID จริงของแกมาแทนที่ด้านล่างน้า):

```env
FACEBOOK_PAGE_ID="ไอดีเพจตัวเลขยาวๆ จาก Step 3"
FACEBOOK_PAGE_ACCESS_TOKEN="คีย์ถาวรขนาดยาวโคตรๆ จาก Step 4"
```

เสร็จสมบูรณ์จ้าแก! ระบบ Veda ของเราจะเชื่อมตรงเข้าเพจเฟสบุ๊คแกเรียบร้อยแล้วจ้า! 🎉🤖🚀
