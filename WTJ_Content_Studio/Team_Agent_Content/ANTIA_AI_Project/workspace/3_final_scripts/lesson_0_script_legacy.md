## [SECTION 1 — ดาวน์โหลดและติดตั้ง — 0:30-2:30]

เข้าไปที่ **antigravity.ai** แล้วคลิก Download

*(แสดงหน้าจอเว็บ)*

---

**Mac:**
1. โหลดไฟล์ `.dmg` → ดับเบิลคลิก → ลากไอคอนไปใส่ Applications → เปิดได้เลย
2. ถ้า Mac ขึ้นเตือนเรื่องนักพัฒนาไม่รู้จัก → **System Settings → Privacy & Security → Open Anyway**

**Windows:**
1. โหลดไฟล์ `.exe` → ดับเบิลคลิก → Next ตลอด → Install
2. ถ้า SmartScreen ขึ้นเตือน → **More Info → Run Anyway**

## [SECTION 3 — ทัวร์ UI — 3:30-6:30]

*(แสดงหน้าจอโปรแกรมตลอด section นี้)*

หน้าตาโปรแกรมมี 3 ส่วนหลัก:

---

**ซ้าย — Projects**  
รายชื่อ Project ทั้งหมดของคุณ  
แต่ละ Project คือ "ห้องทำงาน" แยกกัน — AI แต่ละห้องจำข้อมูลแยกจากกัน  
เช่น Project งานออฟฟิศ กับ Project ธุรกิจส่วนตัว — AI จะรู้คนละเรื่อง

---

**กลาง — Chat**  
ที่นี่คือที่สั่งงาน AI  
ด้านล่างกล่องมีปุ่มให้แนบไฟล์, วาง URL, และเลือก Model ที่จะใช้ (Gemini / Claude / GPT-4o)

---

**ขวา — ผลลัพธ์**  
AI แสดงไฟล์ที่สร้าง, code ที่เขียน, ผลรันโปรแกรม — ทุกอย่างออกมาตรงนี้  
ทำให้คุณเห็นว่า AI กำลังทำอะไรอยู่ตลอดเวลา

---

**GEMINI.md**  
ทุก Project มีไฟล์นี้ซ่อนอยู่  
มันคือกฎที่บอก AI ว่าต้องทำงานแบบไหน โทนไหน มีกฎอะไรบ้าง  
เรื่องนี้จะไปลงลึกใน Lesson 1
