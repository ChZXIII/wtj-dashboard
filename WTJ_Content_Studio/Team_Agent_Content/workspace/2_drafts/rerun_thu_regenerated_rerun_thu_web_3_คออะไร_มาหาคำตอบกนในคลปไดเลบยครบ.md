ทางเลือกที่ 1:
เคยสงสัยไหมว่า Web 1, Web 2, Web 3 มันต่างกันยังไงนะ? หลายคนอาจจะยังไม่รู้ด้วยซ้ำว่าโลกที่เราอยู่มันเปลี่ยนไปแล้ว! น้องนัทจากเพจเช้านี้จะมาเล่าให้ฟังแบบเข้าใจง่ายๆ เลยว่า
- Web 1 คือโลกยุคอ่านอย่างเดียว ไม่ต้องล็อกอิน (คิดถึงกระปุกดอทคอมสมัยก่อน!)
- Web 2 คือโลกที่เราใช้ทุกวัน ล็อกอินด้วยข้อมูลส่วนตัว (อย่าง Facebook, Instagram นี่ไง)
- ส่วน Web 3 คือโลกใหม่ที่ล็อกอินด้วยกระเป๋าบล็อกเชน ไม่สนว่าคุณเป็นใคร ทุกอย่างเริ่มใหม่พร้อมกันหมด ปังเกินต้าน!
แล้วทุกคนคิดว่าการเปลี่ยนผ่านของโลกออนไลน์แบบนี้มันมีผลกับชีวิตเรายังไงบ้าง? มาคุยกันนะจ๊ะ!
