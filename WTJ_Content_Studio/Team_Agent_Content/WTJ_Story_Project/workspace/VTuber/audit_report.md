# ✅ AUDIT REPORT — ถอดรหัส VTuber: ภายใต้หน้ากากอนิเมะ มีอะไรที่โลกความจริงให้ไม่ได้?
ตรวจสอบโดย: ออ | วันที่: 27 พฤษภาคม 2024

สวัสดีค่ะทีมงานทุกคน และน้องครีม... ออตรวจสอบ Research Doc ฉบับปรับปรุงนี้แล้วนะคะ เห็นถึงความตั้งใจในการทำการบ้านเรื่องตัวเลขปี 2024 มากค่ะ ข้อมูลโดยรวมถือว่าแน่นและมีทิศทางที่น่าสนใจ แต่ยังมีบางจุดที่ "ออ" ต้องขอคอมเมนต์เพื่อให้งานของเราออกมาเป๊ะและปลอดภัยที่สุดตามมาตรฐาน WTJ Story ค่ะ

---

## 🟢 ผ่าน (Pass)
- **การนำทฤษฎีมาจับ:** การหยิบยก **The Proteus Effect** (Yee & Bailenson) มาอ้างอิง ถือว่าดีมากค่ะ ทำให้งานมีน้ำหนักทางวิชาการ ไม่ใช่แค่การเขียนตามความรู้สึก
- **Case Study:** กรณีของ **Game部 Project** ข้อมูลถูกต้องตามลำดับเหตุการณ์ และเป็นตัวอย่างที่ชัดเจนที่สุดในการอธิบายความสัมพันธ์ระหว่าง "เปลือก" และ "วิญญาณ"
- **การเปรียบเทียบ:** Analogy เรื่อง **Iron Man** และ **พวงมาลัยดิจิทัล** ช่วยย่อยเนื้อหาที่ซับซ้อนให้เห็นภาพได้ดีเยี่ยม เหมาะกับการนำไปเขียนสคริปต์ต่อค่ะ
- **แหล่งอ้างอิง:** มีการระบุแหล่งที่มาเบื้องต้นชัดเจน ทั้ง User Local และ Playboard

---

## 🔴 ต้องแก้ไข (Fail - Must Fix)
- **ความแม่นยำของขอบเขตสถิติ (User Local):** ครีมระบุว่า "VTuber ทั่วโลกทะลุ 55,000 คน" จาก User Local ออขอให้ตรวจสอบอีกครั้งนะคะ เพราะ User Local ส่วนใหญ่จะ Track ข้อมูลเน้นไปทางฝั่งญี่ปุ่นและ YouTube เป็นหลัก ตัวเลข "ทั่วโลก" จริงๆ (รวม Twitch, Bilibili และ VOMS อื่นๆ) อาจสูงกว่านี้มาก หรือถ้าจะใช้ตัวเลขนี้ **ต้องระบุให้ชัดว่า "เป็นจำนวนที่ได้รับอนุมัติ/ตรวจพบโดย User Local"** เพื่อป้องกันการโต้แย้งเรื่องความครอบคลุมของข้อมูลค่ะ
- **ตัวเลขรายได้ Super Chat:** แม้สถิติ Kiryu Coco จะเป็นตำนาน แต่การระบุตัวเลข 300 ล้านเยน ควรระบุด้วยว่าเป็น **"Gross Revenue" (รายได้ก่อนหักส่วนแบ่ง)** ไม่ใช่เงินที่ศิลปินได้รับจริง เพื่อไม่ให้ผู้ชมเข้าใจผิดเรื่องความรวยของตัวบุคคลค่ะ

## 🟡 ข้อสังเกต (Advisory)
- **ความผันแปรของส่วนแบ่งรายได้:** การคำนวณส่วนแบ่ง (Platform 30% / Agency / Artist) เป็นเพียง "โมเดลคาดการณ์ทั่วไป" อออยากให้โน้ตเพิ่มว่า **"สัดส่วนนี้ขึ้นอยู่กับสัญญาของแต่ละค่าย ซึ่งเป็นความลับทางธุรกิจ"** เพื่อป้องกันปัญหาเรื่องการเปิดเผยข้อมูลที่อาจคลาดเคลื่อนจากความเป็นจริงในบางเอเจนซี่
- **ความล่อแหลมของหัวข้อ Doxxing:** การพูดถึงวิธีสืบหาตัวจริง (เงาสะท้อนในช้อน/เสียงรถ) ต้องระวังอย่าให้กลายเป็นการ "ชี้โพรงให้กระรอก" หรือดูเหมือนเราสนับสนุนพฤติกรรมนั้น ให้เน้นไปที่ **"ผลกระทบทางจิตใจและความปลอดภัยของศิลปิน"** เป็นหลักค่ะ
- **ลิขสิทธิ์ภาพ:** ในพาร์ทความล้มเหลวของ Game部 หากพี่เรย์จะนำไปเขียนภาพประกอบ ต้องระวังการใช้ภาพเปรียบเทียบนักแสดงเก่า-ใหม่ ให้ใช้เป็นเชิงสัญลักษณ์หรือภาพจาก Official ที่ถูกวิพากษ์วิจารณ์เท่านั้นนะคะ
