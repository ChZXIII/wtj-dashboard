แล้วก็ไปคุยกับเค้าไปหาอ่าไปสอบถามหาความ รู้จากเค้าว่าเค้าถ่ายอะไรหาแรงบันดาลใจ อะไรจากเค้าหาแนวการถ่ายของเค้าอะไรอย่าง เงี้ยซึ่งสมัยก่อนเราเราดูรูปเค้าได้จาก ไหนอ่ะดูรูปเค้าได้จากรูปอ่ะรูปเป็นๆน่ะ รูปเป็นหรือที่มันอยู่ในแมกาซีนอะไรอย่าง เงี้ยมากกว่าแต่ทุกวันเนี้ยโอเคมันเป็น โซเชียลเนาะดูจากในอินเทอร์นง อินเทอร์เน็ตได้ง่ายแต่สมัยก่อนเโอ้โห นิทศการอย่างเงี้ยต้องไปดูเทศการภาพถ่าย เนี่ยการประกอบภาพถ่ายของแต่ละยุคเงี้ยโห อันนี้เราไม่นับการแต่งนะเรานับถึงการ ถ่ายรูปหุยทำไมเค้าถ่ายรูปโคตรสวยเลยวะ ใช้โหสีสันโคตรดีเลยวะเมื่อก่อนมันจับ ต้องได้จากเนี่ยอย่างที่พี่กล่าวหนังสือ เอยนิทนศการเอยการเรียนรู้จากบุคคลเอยอ่า แต่สมัยเนี้ย ความรู้เยอะมากเยอะมากแล้วโคตรอิจฉาเด็ก สมัยนี้เลยคือแบบมีอะไรให้แบบเสพเยอะมาก อ่ะทั้งคทentทั้งนั่นทั้งนี่ภาพเคลื่อน ไหวโอ้โหเฉพาะเจาะจงอีกโอ้โหเยอะแยะมาก มายนั้นโอกาสอ่ะในการเรียนรู้อ่ะของของคน ยุคหลังๆอ่ะพี่ถือว่าแบบคุณมีโอกาสเยอะ มากเลยคุณควรจะเ่อหาความรู้เพิ่มเติมให้ เยอะๆเลยทุกวันนี้ผมก็ยังยังศึกษาความรู้ ในในใน วงการอีกอยู่เราพยายามทำตัวไม่เป็นน้ำ เต็มแก้วอ่ะพูดง่ายๆอ่ะมีอะไรใหม่ๆอ่าอ่า สมัยนี้เค้านิยมอะไรอ่าทริกต่างๆนานาเรา ก็ไปดูอ่าแล้วก็เราก็ไปประยุกต์ให้เป็น งานส่วนตัวเราด้วยอะไรอย่างเงี้ยครับใช่ >> ครับพี่รักในอาชีพทางภาพงานแต่งพี่ให้ เปรียบเปยให้แบบอะไรยังไง >> โอถามว่าทุกวันเนี้ยรักนะรักเลยแหละรักใน อาชีพตัวเองก่อนอ่าแล้วรักในสาย ที่ตัวเองทำด้วยอ่ารักอาชีพก็คืออาชีพ ช่างภาพถูกมั้ยรักษาก็คืออ่าสายช่างภาพ เวดดิ้งอ่ารักๆรักเลยแหละเพราะว่ามันมัน ไม่ได้แค่ทำ รายได้กับเราแต่มันคือเหมือนคล้ายๆว่าเรา ทำมา 20 ปีแล้วอ่ะเราทำมันให้มันเหมือน แบบมันครึ่งชีวิตของเราแล้วอ่ะมันอยู่กับ เราไปแล้วอ่ะอะไรอย่างเงี้ยดังนั้น แนที่ที่บอกก็คือเป็นส่วนหนึ่งของชีวิต อ่ะมันคือวันหนึ่งให้เราแยกสายทำอย่าง อื่นหรอ มันก็เหมือนเริ่มใหม่หมดอ่ะดังนั้นช่าง ภาพสำหรับผมมันคือชีวิตผมละอ่ามันคือ ฮีลใจเราด้วย สร้างรายได้ให้เราด้วยแล้วก็ อ่ายังมีความท้าทาย ของมันน่ะตลอดดังนั้นถามว่ารักมั้รัก และอีกอย่างที่รักเสมอไม่เปลี่ยนแปลงรัก การศิลปะ >> พี่คิดว่าพี่จะทำงานนี้ไปได้ไปกี่ปี >> เฮ้ยยังมีโอโหจะบอกเลยว่าฝรั่งอ่ะบางคน น่ะ  60  70  80 ยังทำอยู่เลยดังนั้นเราอย่า ทำตัวให้แก่ดิเราเราต้องมีพาเวอร์คือถาม ว่าทำเท่าที่แรงเราทำได้เลยผมจะจะไม่มี เกษียณอ่ะบางคนบอกเกษียณ 50  60  จะไปไปผูกกรอบตัวเองทำไมอ่ะถูกป่ะเราคน เรามันโดนตีกรอบไงอายุ 60 เกษียณอายุนั่น มันเรื่องของคนอื่นสมมุติผมอ่ะผมมีแรง เท่าไหร่ผมก็ทำไปเออโอเคแบบอย่างน้อยอ่ะ อ่ามันอาจจะลดลดความเข้มข้นลงบ้างตาม เรื่องเรื่องด้วยวัยเรื่องด้วยของอ่าอะไร ร่างกายอย่างเงี้ยแต่ถามว่าจะให้หยุดเหรอ มันมันไม่มีเกษียณผมไม่มีเกษียณเพียงแต่ ว่าเราก็อาจก็ตามยุคตามสมัยอ่า [เสียงกระแอม]ช่วงนี้อะไรมาช่วงนี้อะไร เราก็ต้องเราก็ต้องตามเทรนด้วยแหละเออก็ ทุกอย่างมันก็จะเป็นเทรนเนาะตามแฟช่ง แฟชั่นอะไรมันก็จะเป็นเทรนศิลปะอะไรทุก อย่างก็เป็นเทรนแม้กระทั่งการถ่ายรูปมัน ก็จะเป็นเทรนมันไม่ใช่แบบโอ้โหอ่าสไตล์ นี้ไปจนถึง 30-40 ปีมันก็ไม่ใช่อะไรอย่าง เงี้ยแต่สิ่งที่อย่างที่ผมบอกตั้งแต่ต้น เลยมาตรฐานใช่ >> สุดท้ายก็ไม่มีละฝากฟ้าติดปากผลงานที่จะ เล่นที่แบบนี้ >> อ่า[เสียงกระแอม]ถ้าถ้าอยากตามผลงานก็ตาม ในเพจเลยครับบวกเก่งโตฟเฟอร์เลยครับ >> ไอ้เดี๋ยวถ่ายเบื้องหลังให้พี่หน่อยพี่ไป อัพ[เพลง]ให้เดี๋ยว >> ไม่เป็นไรเอาถ่ายกล้องพี่ก็ได้ถ่ายเบื้อง หลังตอนแบบเนี้ยงี้ก็ได้เออให้
