## 🔑 3. ตู้เก็บกุญแจระบบ (Credentials & Keys Cabinet)
*(ตั้งอยู่ที่โฟลเดอร์: `./credentials/`)*
- **รายละเอียด:** เป็นห้องนิรภัยที่รวบรวมไฟล์สิทธิ์การเข้าถึง (API Tokens, Client Secrets, Trade States) ทั้งหมดในระบบ ไม่ว่าจะเป็น:
  - `token_sheets.json` (Google Sheets Token สำหรับน้องเจนและน้องวิน)
  - `client_secret.json` (Google API Secrets)
  - `token_youtube.json` (YouTube Token สำหรับน้องเรย์)
  - `token_tiktok.json.bak` (TikTok Backup Token)
  - `trade_sync_state.json` (สถานะการซิงค์พอร์ตคริปโตของน้องวิน)
- **กฎเหล็ก:** ห้ามนำไฟล์กุญแจเหล่านี้ออกมาวางเกะกะนอกตึก (รูทโฟลเดอร์) เด็ดขาด สคริปต์ทุกตัวของทีมงานต้องวิ่งเข้ามาเรียกใช้ไฟล์จากห้อง `credentials/` นี้เท่านั้น!

## 🎬 4. ทีมงานผลิตคอนเทนต์ (Content Team)
*(ตั้งอยู่ที่โฟลเดอร์: `./Team_Content_Studio/`)*

ทีมงาน Content แบ่งออกเป็น 2 ห้องทำงานเอกเทศอย่างชัดเจน:

### 🏠 ห้องทำงานหลัก (WTJ & Photo)
*(ตั้งอยู่ที่โฟลเดอร์: `./Team_Content_Studio/Team_Agent_Content/`)*
รับผิดชอบดูแล 3 รายการหลักของเก่ง:
1. **WTJ Podcast / Talk:** รายการสัมภาษณ์พูดคุยกับแขกรับเชิญ ดำเนินรายการโดยพี่เก่ง ออนแอร์ทุกวันพฤหัสบดี เวลา 20:00 น.

3. **Photography Project:** โครงการระบบออโต้โพสต์ภาพถ่ายสะสมพอร์ตโฟลิโอสำหรับเพจถ่ายภาพอาหาร และเพจภาพบุคคล/เช่าสตูดิโอ [📸 ดูรายละเอียดห้องทำงาน](Team_Content_Studio/Team_Agent_Content/Photography_Project/README.md)
