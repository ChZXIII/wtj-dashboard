# GHN168 Dashboard & Document Sync App 📊✨

ยินดีต้อนรับสู่แอปพลิเคชันคลังข้อมูลบัญชีและจัดทำเอกสารของ **บริษัท จีเอชเอ็น 168 มีเดีย แอนด์ ครีเอชั่น จำกัด** 

แอปพลิเคชันนี้ทำงานเป็น **Static Web App** 100% (HTML/CSS/JS) ปลอดภัย ไม่มีเซอร์วิสรันเบื้องหลังเพื่อประหยัดแบตเตอรี่และ CPU ของเครื่อง Mac สามารถดับเบิ้ลคลิกเพื่อใช้งานได้ทันทีออฟไลน์ หรือต่ออินเทอร์เน็ตเพื่อยิงข้อมูลบัญชีลง Google Sheets

---

## 📂 โครงสร้างโฟลเดอร์ของโปรเจกต์
- `index.html` - หน้าแผงควบคุมหลักและฟอร์มสร้างเอกสาร
- `style.css` - สไตล์การออกแบบธีม **3D Solid Shadow (Retro Brutalist)** ในโทน ดำ-ขาว-แดงแอคเซนท์
- `app.js` - สคริปต์การคำนวณทางบัญชี (VAT 7%, WHT 1%/3%/5%), การเก็บประวัติท้องถิ่น และการซิงค์ข้อมูล
- `google_sheets_sync_script.gs` - โค้ด Apps Script สำหรับเอาไปวางลงใน Google Sheets ของบริษัท
- `README.md` - คู่มือการใช้งานเล่มนี้แก

## 🛠️ ขั้นตอนเชื่อมโยงระบบกับ Google Sheets (ทำครั้งเดียว)

เพื่อให้แอปสามารถยิงยอดบัญชีส่งตรงไปลงชีตส่วนกลางของบริษัทได้ ให้แกทำตามขั้นตอนนี้จ้า:

### ขั้นตอนที่ 1: ติดตั้ง Apps Script ใน Google Sheets
1. เปิดแผ่นงาน **Google Sheets** ของบริษัท GHN168 ที่ต้องการให้เป็นฐานข้อมูลรับเงินเข้า-ออก
2. กดเมนูบาร์ด้านบน: **ส่วนขยาย (Extensions)** > **Apps Script**
3. ลบโค้ดที่มีอยู่ในนั้นออกทั้งหมด แล้วคัดลอกโค้ดจากไฟล์ `google_sheets_sync_script.gs` ในโฟลเดอร์นี้ไปวางแทนที่
4. กดเซฟโครงการ (รูปแผ่นดิสก์ หรือกด `Cmd + S` บน Mac)

### ขั้นตอนที่ 2: เปิดการติดตั้งใช้งาน (Deploy)
1. กดปุ่มสีน้ำเงินด้านบนขวาเลือก **การใช้งานได้จริง (Deploy)** > **การติดตั้งใช้งานใหม่ (New deployment)**
2. คลิกรูป **ฟันเฟือง (Gear)** ข้างหัวข้อเลือกประเภท เลือก **เว็บแอป (Web app)**
3. ตั้งค่าการดีพลอยดังนี้:
   - **คำอธิบาย:** ระบบบัญชี GHN168 v1.0
   - **เรียกใช้ในฐานะ (Execute as):** เลือก **ฉัน (อีเมลบัญชี Gmail ของบริษัทแกเอง)**
   - **ผู้มีสิทธิ์เข้าถึง (Who has access):** เลือก **ทุกคน (Anyone)** *(ต้องเลือกเป็น Everyone เพื่อให้หน้าเว็บบราวเซอร์ยิงมาคีย์ค่าได้)*
4. กดปุ่ม **การติดตั้งใช้งาน (Deploy)** สีน้ำเงิน
5. หาก Google มีหน้าต่างเตือนเรื่องความปลอดภัย ให้กดให้สิทธิ์อนุญาตการใช้งานให้เรียบร้อยจ้า
6. คัดลอก **URL ของเว็บแอป (Web app URL)** (ลิงก์จะลงท้ายด้วย `/exec`) เก็บเอาไว้

### ขั้นตอนที่ 3: ตั้งค่าเชื่อมต่อในแอป
1. ดับเบิ้ลคลิกเปิดไฟล์ `index.html` ของโปรเจกต์ GHN168 ในบราวเซอร์
2. เลือกเมนูด้านซ้าย: **การตั้งค่าเชื่อมต่อ (Settings)**
3. วางลิงก์ที่ก๊อปปี้มาในช่อง **Google Apps Script Web App URL**
4. เอา ID ของ Google Sheets (ตัวเลขยาวๆ ใน URL หน้าเว็บของชีตหลังช่วง `/d/` และก่อน `/edit`) มากรอกลงในช่อง **Google Spreadsheet ID**
5. กดปุ่ม **บันทึกการเชื่อมต่อบัญชี** สีส้ม
6. เรียบร้อยแก! ระบบพร้อมทำงานแล้วจ้า

## 📊 ฟังก์ชันการทำงานและการออกเอกสาร
แอปนี้ได้รับการตรวจสอบความถูกต้องโดย **น้องพิม (Pim)** เอเจนต์บัญชีและการเงิน มีฟังก์ชันดังนี้:

1. **ระบบเอกสารทั่วไป (QT/IV/RE):** 
   - รองรับการป้อนรายการบริการ คำนวณราคาย่อย ยอดรวม VAT 7% และหัก ณ ที่จ่าย (WHT) อัตโนมัติ พร้อมแปลงตัวเลขเป็นตัวหนังสือภาษาไทย
   - เมื่อออก **ใบเสร็จรับเงิน (Receipt)** และกดบันทึก ข้อมูลจะซิงค์ไปที่ชีตแท็บ `รายรับ` ทันที
2. **ระบบ 50 ทวิ (WHT):**
   - ออกใบรับรองภาษีหัก ณ ที่จ่าย 50 ทวิ สำหรับจ่ายค่าบริการ/จ้างทำของให้กับฟรีแลนซ์หรือคู่ค้า
   - เมื่อบันทึก ข้อมูลจะซิงค์ไปที่ชีตแท็บ `รายจ่าย` ทันที
3. **ระบบความปลอดภัยของเงินบริษัท (HITL Audit by Pim):**
   - ทุกครั้งที่มียอดโอนออกเกิน **10,000 บาท** ระบบจะขึ้นกรอบคำเตือนสีแดงเตือนแกทันที และบังคับให้แกหรือพี่เฟิสคลิกยืนยันการทำรายการก่อน จึงจะกดบันทึกลงฐานข้อมูลชีตได้จ้า
4. **คลังเอกสารสำคัญ (Document Hub):**
   - ใช้บันทึกเก็บลิงก์ทางด่วน Google Drive ของไฟล์สำคัญในบริษัทเพื่อให้ทุกคนคลิกหาเจอง่ายๆ จากที่นี่ที่เดียว
