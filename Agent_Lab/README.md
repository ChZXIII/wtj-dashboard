ยินดีต้อนรับเข้าสู่ **ห้องทดลองระบบ (Agent Lab)** ครับพี่เก่ง! 🦾
ห้องนี้ถูกจัดตั้งขึ้นโดย **น้องคิว (Q)** เพื่อใช้เป็นศูนย์รวมการทดสอบสคริปต์อัตโนมัติ รันระบบทดลอง และเก็บรักษาเคสบั๊กที่สำคัญเพื่อการเรียนรู้หลังบ้าน โดยไม่ทำให้พื้นที่หลักของออฟฟิศรกรุงรังครับ!

## 📂 โครงสร้างภายในห้องทดลอง (Lab Directory Structure)

- `Agent_Lab/sandbox/` ➡️ พื้นที่สำหรับการรันไฟล์สคริปต์ทดสอบชั่วคราว ดึงข้อมูลทดลอง หรือจำลองสถานการณ์
- `Agent_Lab/sandbox/flowise/` ➡️ พื้นที่เก็บข้อมูลฐานข้อมูล SQLite, คีย์ และคอนฟิกของระบบบอร์ด Flowise
- `Agent_Lab/logs/` ➡️ จุดเก็บประวัติไฟล์ Log ที่เกิดจากการเทสระบบทั้งหมด (เพื่อป้องกันไม่ให้ Log ลอยไปเกะกะโฟลเดอร์หลัก)
- `Agent_Lab/bug_vault/` ➡️ คลังเก็บตัวอย่างโค้ดหรือสคริปต์ที่มี Bug / เคสพิเศษที่เคยพัง เพื่อใช้เป็นบทเรียนศึกษาแนวทางป้องกันในอนาคต

---

## 📝 กฎเหล็กประจำห้องทดลอง (Lab Guidelines)

1. **Test inside Sandbox:** เวลาต้องการเขียนสคริปต์ทดสอบระบบ หรือรันไฟล์ทดลองใดๆ ให้ทำภายในห้อง `Agent_Lab/sandbox/` เสมอ
2. **Log to Lab Logs:** สคริปต์รันเทสทุกตัว ต้องถูกตั้งค่าชี้พาธการบันทึก Log ไฟล์มาที่ `Agent_Lab/logs/`
3. **Register Bugs in Vault:** หากสคริปต์ตัวไหนรันแล้วเจอ Bug พิเศษหรือ Error ที่น่าสนใจ ให้ย้ายไฟล์นั้นมาเก็บไว้ที่ `Agent_Lab/bug_vault/` พร้อมเขียนไฟล์คำอธิบายสั้นๆ (เช่น `README.md` หรือแนบข้อความคอมเมนต์หัวไฟล์) เพื่อเป็นคลังความรู้ให้ทีมงานและพี่เก่งเข้ามาศึกษา
4. **Clean up Root:** ห้ามปล่อยให้ไฟล์ผลลัพธ์จากการรันเทสหรือไฟล์ขยะใดๆ ลอยออกไปนอกเขตห้องทดลองเด็ดขาด!
